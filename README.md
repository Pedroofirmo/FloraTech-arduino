# 🌱 FloraTech - Monitoramento Inteligente de Estufas (Arduino)

---

## 🚀 Sobre o Projeto

O **FloraTech** é um sistema inteligente desenvolvido para monitorar estufas de plantas, trazendo mais controle, eficiência e automação para o cultivo.

O projeto utiliza **Arduino + sensores ambientais** para coletar dados em tempo real e auxiliar na tomada de decisões.

---

## 🎯 Objetivo

Garantir condições ideais para o crescimento das plantas, monitorando variáveis essenciais e reduzindo perdas na produção.

---

## ⚙️ Funcionalidades

* 🌡️ Monitoramento de temperatura
* 💧 Umidade do ambiente
* 🌱 Umidade do solo
* 🧪 Medição de pH do solo 
* 💨 Nível de CO₂ 
* 💡 Luminosidade

## 🔌 Componentes Utilizados

* Arduino 
* Sensor de temperatura/umidade (DHT11 ou DHT22)
* Sensor de umidade do solo
* Sensor de luminosidade (LDR)
* Sensor de pH 
* Sensor de CO₂ 
* Jumpers e protoboard

---

## 🧠 Como Funciona

1. Os sensores coletam dados do ambiente
2. O Arduino processa as informações
3. Os dados podem ser exibidos no serial monitor ou enviados para um sistema externo
4. Futuramente, o sistema poderá acionar alertas ou automações
